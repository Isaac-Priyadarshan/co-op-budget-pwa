# Co-Op Budget PWA

A premium private 2-person budget and finance management PWA for Isaac & Jenifa.

## Tech Stack
- React 19 + TypeScript + Vite 8
- Tailwind CSS v4 + Framer Motion 12
- Supabase (PostgreSQL backend)
- Vercel (hosting + auto-deploy)

## Features
- 12 screens across 4 navigation groups
- Premium dark / pitch-black theme with glassmorphism
- Fluid animations, mobile-first UX
- No traditional login — session-only user identity engine

## Screens
**Group 1 — Finance:** Budget · Home · Ledger  
**Group 2 — Tracking:** Borrowed · Wallet & Credit · Lent  
**Group 3 — Assets:** Loans · Account Overview · Asset  
**Group 4 — More:** Recurring Payment · Overview · Settings
